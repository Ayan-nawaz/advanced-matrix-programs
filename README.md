# Advanced Matrix Programs

A collection of C programs demonstrating advanced matrix operations and transformations. This repository is designed for beginners and intermediate learners to practice common 2D array and matrix manipulation techniques in C.

## Programs Included

1. **zero-one.c**
   - Processes and displays zero and one elements in a matrix.

2. **transpose.c**
   - Computes and displays the transpose of a matrix.

3. **transpose-in-place.c**
   - Performs in-place transpose of a square matrix without using extra memory.

4. **rotate-matrix.c**
   - Rotates a matrix by 90 degrees.

5. **multiplication.c**
   - Performs multiplication of two matrices and displays the result.

## Topics Covered
